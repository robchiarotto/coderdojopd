TAG DESCRIZIONE TAG DESCRIZIONE TAG DESCRIZIONE
<!--...--> Per inserire un commento nel codice HTML <!DOCTYPE>  Va utilizzato per definire il tipo di documento <a> Collegamento ipertestuale
<abbr> Va utilizzato per definire un'abbreviazione o un acronimo <address> Va utilizzato per definire contact information for the author/owner of a document <area> Va utilizzato per definire an area inside an image-map
<article> Per definire un testo che deve essere considerato un articolo <aside> Va utilizzato per definire content aside from the page content <audio> Va utilizzato per definire sound content
<b> Testo in grassetto <base> Va utilizzato per definire un URL/percorso di default per tutti gli URL/indirizzi nella pagina <bdi> Va utilizzato per isolare una parte di testo che potrebbe essere formattata in una direzione differente rispetto al testo fuori dal tag
<bdo> Cambia la direzione del testo <blockquote> Per definire una sezione di testo "quotata" e quindi ripresa da un'altra parte <body> Va utilizzato per definire il corpo del documento
<br> Va utilizzato per definire una linea di break (va a capo) <button> Va utilizzato per definire un bottone cliccabile <canvas> Used to draw graphics, on the fly, via scripting (usually JavaScript)
<caption> Va utilizzato per definire a table caption <cite> Va utilizzato per definire the title of a work <code> Va utilizzato per definire un pezzo di codice all'interno dell'HTML
<col> Specifies column properties for each column within a <colgroup> element  <colgroup> Specifies a group of one or more columns in a table for formatting <datalist> Specifies a list of pre-defined options for input controls
<del> Usato per definire un pezzo di test che è stato eliminato dal documento (testo barrato) <details> Va utilizzato per definire additional details that the user can view or hide <dfn> Represents the defining instance of a term
<dialog> Va utilizzato per definire una finestra di dialog <div> Va utilizzato per definire una sezione all'interno della pagina <dl> Va utilizzato per definire una lista con delle descrizioni
<dt> Va utilizzato per definire un termine all'interno della lista ><dl> <dd> Va utilizzato per definire la descrizione di un termine all'interno di una lista ><dl> <em> Va utilizzato per definire un testo in italico
<embed> Va utilizzato per definire a container for an external (non-HTML) application <fieldset> Groups related elements in a form <figcaption> Va utilizzato per definire a caption for a <figure> element
<figure> Specifies self-contained content <footer> Va utilizzato per definire un footer (piè di pagina) per il documento o per la sezione <form> Va utilizzato per definire un form HTML dove l'utente può inserire dei dati in input
<h1> to <h6> Va utilizzato per definire HTML headings <head> Va utilizzato per definire information about the document <header> Va utilizzato per definire a header for a document or section
<hr> Va utilizzato per definire a thematic change in the content <html> Va utilizzato per definire the root of an HTML document <i> Va utilizzato per definire a part of text in an alternate voice or mood
<iframe> Va utilizzato per definire an inline frame <img> Va utilizzato per definire an image <input> Va utilizzato per definire an input control
<ins> Va utilizzato per definire a text that has been inserted into a document <kbd> Va utilizzato per definire keyboard input <keygen> Va utilizzato per definire a key-pair generator field (for forms)
<label> Va utilizzato per definire a label for an <input> element <legend> Va utilizzato per definire a caption for a <fieldset> element <li> Va utilizzato per definire a list item
<link> Va utilizzato per definire the relationship between a document and an external resource (most used to link to style sheets) <main> Specifies the main content of a document <map> Va utilizzato per definire a client-side image-map
<mark> Va utilizzato per definire marked/highlighted text <menu> Va utilizzato per definire a list/menu of commands <menuitem> Va utilizzato per definire a command/menu item that the user can invoke from a popup menu
<meta> Va utilizzato per definire metadata about an HTML document <meter> Va utilizzato per definire a scalar measurement within a known range (a gauge) <nav> Va utilizzato per definire navigation links
<noscript> Va utilizzato per definire an alternate content for users that do not support client-side scripts <object> Va utilizzato per definire an embedded object <ol> Va utilizzato per definire an ordered list
<optgroup> Va utilizzato per definire a group of related options in a drop-down list <option> Va utilizzato per definire an option in a drop-down list <output> Va utilizzato per definire the result of a calculation
<p> Va utilizzato per definire a paragraph <param> Va utilizzato per definire a parameter for an object <picture> Va utilizzato per definire a container for multiple image resources
<pre> Va utilizzato per definire preformatted text <progress> Represents the progress of a task <q> Va utilizzato per definire a short quotation
<rp> Va utilizzato per definire what to show in browsers that do not support ruby annotations <rt> Va utilizzato per definire an explanation/pronunciation of characters (for East Asian typography) <ruby> Va utilizzato per definire a ruby annotation (for East Asian typography)
<s> Va utilizzato per definire text that is no longer correct <samp> Va utilizzato per definire sample output from a computer program <script> Va utilizzato per definire a client-side script
<section> Va utilizzato per definire a section in a document <select> Va utilizzato per definire a drop-down list <small> Va utilizzato per definire un testo più piccolo
<source> Va utilizzato per definire multiple media resources for media elements (<video> and <audio>) <span> Va utilizzato per definire a section in a document <strong> Va utilizzato per definire important text
<style> Va utilizzato per definire style information for a document <sub> Va utilizzato per definire subscripted text <summary> Va utilizzato per definire a visible heading for a <details> element
<sup> Va utilizzato per definire superscripted text <table> Va utilizzato per definire a table <tbody> Va utilizzato per definire un gruppo di colonne del corpo in una tabella
<td> Va utilizzato per definire una singola cella in una tabella <textarea> Va utilizzato per definire una text-area, cioè un input di test multi-linea <tfoot> Va utilizzato per definire un gruppo di colonne footer (piè di pagina) in una tabella
<th> Va utilizzato per definire la cella d'intestazione di una colonna in una tabella <thead> Va utilizzato per definire un gruppo di colonne testata in una tabella <time> Per definire la data e l'ora
<title> Va utilizzato per definire il titolo della pagina web <tr> Va utilizzato per definire una riga in una tabella <track> Va utilizzato per definire text tracks for media elements (<video> and <audio>)
<u> Testo sottolineato <ul> Va utilizzato per definire una lista non ordinata (elenco puntato) <var> Utilizzato per definire una variabile
<video> Per inserire un video <wbr> Va utilizzato per definire un a capo che verrà applicato solo se il testo lo richiede
